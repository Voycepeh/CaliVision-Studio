# Drill Exchange Vision

## Vision summary

Drill Exchange is the planned shared package platform inside the Studio ecosystem.

It is intended to provide:

- account ownership,
- hosted package storage,
- discovery and sharing,
- versioned publishing,
- fork/remix flows,
- simplified collaboration/update model.

## Product goals

- let users keep a personal library of owned drill packages,
- enable sharing and discovery of reusable drills,
- make package lineage explicit and understandable,
- support version updates without breaking mobile import expectations,
- preserve Studio as authoring source of truth.

## Non-goals (current)

- claiming fully shipped marketplace/commercial features,
- claiming complete hosted moderation/governance systems today,
- replacing mobile runtime responsibilities.

## Relationship to mobile runtime

Shared or published packages remain consumable by the mobile runtime client after import.

Android repo: <https://github.com/Voycepeh/CaliVision>.
