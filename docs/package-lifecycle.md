# Package Lifecycle

1. Start from Library home and open Drill Studio.
2. Author/edit drill and phases.
3. Export a portable package artifact.
4. Optionally mock-publish to local Drill Exchange listing.
5. Import/use in mobile runtime client.
6. Fork/remix/version and republish over time.

Android runtime client reference: <https://github.com/Voycepeh/CaliVision>.
